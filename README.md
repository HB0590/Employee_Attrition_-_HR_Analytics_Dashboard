# Employee_Attrition_-_HR_Analytics_Dashboard
Excel HR analytics workbook (300 employees) using VLOOKUP, INDEX/MATCH, IFS, AND, IFERROR, COUNTIFS, AVERAGEIFS, TEXTJOIN, DATEDIF, EOMONTH, MEDIAN, STDEV, PERCENTILE, RANK, SUMPRODUCT, data validation, and conditional formatting to analyze headcount, attrition, retention risk, and salary distribution.
